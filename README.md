Sistema Cadastral Web com Java EE
Projeto desenvolvido para a disciplina DGT2811 - Desenvolvimento Back-End Corporativo com Java e Cloud.
Descrição
Aplicação Web para cadastro de produtos utilizando Java EE (Jakarta EE), JPA, EJB, Servlets, JSP, SQL Server e Bootstrap. O sistema implementa operações CRUD (Cadastrar, Listar, Alterar e Excluir), aplicando conceitos de persistência de dados, regras de negócio e desenvolvimento Web corporativo.
Objetivos
    • Implementar persistência de dados com JPA. 
    • Aplicar regras de negócio utilizando EJB. 
    • Desenvolver interfaces Web com Servlets e JSP. 
    • Melhorar o design com Bootstrap. 
    • Integrar aplicação Java Web ao SQL Server. 
Tecnologias Utilizadas
    • Java (JDK) 
    • NetBeans IDE 
    • Jakarta EE 8 
    • GlassFish Server 
    • SQL Server 
    • JPA 
    • EJB 
    • Servlets e JSP 
    • Bootstrap 5 
Funcionalidades
    • Cadastro de produtos 
    • Listagem de produtos 
    • Alteração de produtos 
    • Exclusão de produtos 
    • Interface responsiva com Bootstrap 
Autor
Projeto acadêmico desenvolvido para fins educacionais na disciplina DGT2811 – Desenvolvimento Back-End Corporativo com Java e Cloud.

# Sistema Cadastral Web com Java EE

## 📌 Sobre o Projeto

Este projeto foi desenvolvido para a disciplina **DGT2811 - Desenvolvimento Back-End Corporativo com Java e Cloud**.

Trata-se de uma aplicação Web para gerenciamento e cadastro de produtos utilizando tecnologias Java EE (Jakarta EE), permitindo realizar operações completas de CRUD (**Create, Read, Update e Delete**), com integração a banco de dados SQL Server.

O sistema foi construído seguindo os conceitos de desenvolvimento corporativo, separando as camadas de persistência, regras de negócio e interface Web.

---

## 🎯 Objetivos do Projeto

* Implementar persistência de dados utilizando JPA.
* Aplicar regras de negócio através de EJB.
* Desenvolver interfaces Web utilizando Servlets e JSP.
* Melhorar a experiência do usuário com Bootstrap.
* Integrar uma aplicação Java Web ao banco de dados SQL Server.
* Praticar conceitos de desenvolvimento Back-End corporativo.

---

## 📂 Estrutura do Projeto

O projeto segue uma arquitetura em camadas:

### Persistência

Responsável pela comunicação com o banco de dados através do JPA.

### Regras de Negócio

Implementadas utilizando EJB para centralizar as operações da aplicação.

### Controle

Realizado por Servlets, responsáveis por receber as requisições dos usuários.

### Apresentação

Construída com JSP e Bootstrap para oferecer uma interface amigável e responsiva.

---

## ⚙️ Funcionalidades

### Cadastro de Produtos

Permite inserir novos produtos no sistema.

### Listagem de Produtos

Exibe todos os produtos cadastrados.

### Alteração de Produtos

Permite atualizar informações de produtos existentes.

### Exclusão de Produtos

Remove produtos do banco de dados.

### Interface Responsiva

Utiliza Bootstrap para adaptação em diferentes dispositivos.

---

## 🗄 Banco de Dados

O sistema utiliza o **Microsoft SQL Server** para armazenamento das informações dos produtos.

A persistência é realizada através do **JPA**, facilitando o mapeamento objeto-relacional (ORM).

---

## 🚀 Como Executar o Projeto

### Pré-requisitos

* Java JDK instalado
* Apache NetBeans IDE
* GlassFish Server configurado
* SQL Server instalado e em execução

### Passos

1. Clone ou baixe o projeto.
2. Abra o projeto no NetBeans.
3. Configure a conexão com o SQL Server.
4. Inicie o servidor GlassFish.
5. Execute o projeto pelo NetBeans.
6. Acesse a aplicação pelo navegador.
